Aproximación normal

Un equipo tradicional de operaciones escala linealmente con el tamaño del servicio.

Lo que implica contratar más personas para hacer las mismas tareas una y otra vez.

La intervencion manual para cambios y manejo de eventos se vuelve más costosa a medida que crece el servicio.

Aproximación SRE

Busca que:

50% Dev

50% Ops

Alta dependencia de automatización vs esfuerzo humano.

Estos equipos escalan de manera sublineal con el tamaño del servicio.

Se puede ver a SRE como una implementación especifica de DevOps con algunas extensiones
